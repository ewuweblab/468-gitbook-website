# Practice Pens

### Process

1. Define a need or want e.g. I want confetti!
2. Search "JavaScript library _insert Your goal_
3. Look at demos, and read API documentation
4. Create a _CodePen Pen_
5. What the requirements for:
   * HTML
   * CSS
   * JS
